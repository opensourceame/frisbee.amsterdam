## Contributing to frisbee.amsterdam

<p>This page was generated by <a href="https://pages.github.com">GitHub Pages</a>.</p>

Do this:

    git clone git@github.com:opensourceame/frisbee.amsterdam.git

Testing locally:

```sh
gem install bundler
# bundle install
bundle exec jekyll serve -o &> log &
```
